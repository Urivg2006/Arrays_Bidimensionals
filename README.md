# ☕ Arrays Bidimensionals i Mètodes en Java

Benvingut/da a aquest repositori. Aquest repositori conté una col·lecció de programes Java que demostren l'ús de matrius bidimensionals (**Arrays 2D**) i la modularitat mitjançant la creació de **mètodes** (funcions). 

Cada programa aborda un cas d'ús diferent per il·lustrar com es poden utilitzar les matrius per emmagatzemar dades en files i columnes, així com l'estructuració del codi en blocs reutilitzables per millorar-ne la claredat i l'eficiència.

![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)

## 📂 Programes inclosos:

* **`Array2D.java`**:
    * Gestor de notes acadèmiques mitjançant un menú interactiu. Utilitza un array bidimensional per a les notes i un d'unidimensional per als noms dels grups, permetent calcular mitjanes, notes màximes i comptar el total d'alumnes aprovats.
* **`Arraybi1.java`**:
    * Crea i imprimeix per pantalla una matriu de 10x10, omplint-la de manera seqüencial amb números de l'1 al 100 en format de graella.
* **`Arraybi2.java`**:
    * Genera una matriu de 5x5 on els valors s'omplen aplicant un patró específic, utilitzant un desplaçament matemàtic (restrant 4) a l'inici de cada nova fila.
* **`Arraybi3.java`**:
    * Inicialitza una matriu de 4x6 amb valors aleatoris (entre 0 i 200). Recorre les dades per trobar i mostrar el valor màxim, el mínim i la suma total de tots els elements.
* **`Arraybi4.java`**:
    * Genera una matriu quadrada amb una mida (`n x n`) definida per l'usuari i l'omple de nombres aleatoris. Després de mostrar-la, demana un número i compta exactament quantes vegades apareix dins la graella.
* **`FuncionsArrays.java`**:
    * Mostra l'ús pràctic de mètodes independents (`funcio1` a `funcio4`) operant sobre arrays unidimensionals: inclou generació seqüencial, introducció de dades per teclat, emplenament amb aleatoris i cerca de freqüències.
* **`JavaZerosMetodes.java`**:
    * Divertit minijoc de consola on l'usuari ha de trobar 5 "zeros" amagats en un taulell de 5x5. El jugador disposa de 8 intents i el taulell s'actualitza visualment (amb colors a la terminal) marcant els encerts ('E') i les fallades ('D'). Destaca per un ús avançat del pas de matrius a través de funcions (`inicialitzar`, `generarZeros`, `jugar`, `imprimir`).

**English:**

# ☕ 2D Arrays and Methods in Java

Welcome to this repository. This collection contains Java programs designed to demonstrate the use of two-dimensional arrays (**2D Arrays**) and modular programming through the creation of **methods** (functions). 

Each program addresses a different use case to illustrate how matrices can be used to store data in rows and columns, as well as structuring code into reusable blocks to improve clarity and efficiency.

## 📂 Included Programs

* **`Array2D.java`**:
    * An academic grade manager using an interactive menu. It uses a 2D array for student grades and a 1D array for group names, allowing calculations for averages, maximum grades, and counting passed students.
* **`Arraybi1.java`**:
    * Creates and prints a 10x10 matrix, filling it sequentially with numbers from 1 to 100 in a grid format.
* **`Arraybi2.java`**:
    * Generates a 5x5 matrix populated with a specific numerical pattern, applying a mathematical offset (subtracting 4) at the beginning of each new row.
* **`Arraybi3.java`**:
    * Initializes a 4x6 matrix with random values (between 0 and 200). It traverses the data to find and display the maximum value, the minimum value, and the total sum of all elements.
* **`Arraybi4.java`**:
    * Generates a user-defined square matrix (`n x n`) filled with random numbers. After displaying it, it prompts the user for a specific number and counts exactly how many times it appears within the grid.
* **`FuncionsArrays.java`**:
    * Demonstrates the practical use of independent methods (`funcio1` to `funcio4`) working on one-dimensional arrays. It includes sequential generation, keyboard data entry, random population, and frequency searching.
* **`JavaZerosMetodes.java`**:
    * A fun console mini-game where the user must find 5 hidden "zeros" on a 5x5 board. The player has 8 attempts, and the board state updates visually (with terminal colors) marking hits ('E') and misses ('D'). It stands out for its advanced use of passing matrices through functions (`inicialitzar`, `generarZeros`, `jugar`, `imprimir`).
